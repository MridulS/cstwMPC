the folder contains files for 
cstKS  project (longer WP version available at: https://www.ecb.europa.eu/pub/pdf/scpwps/ecbwp1633.pdf) and 
cstwMPC project (longer WP version available at: https://www.ecb.europa.eu/pub/pdf/scpwps/ecbwp1655.pdf)


- "NoAggShock" folder contains various files which produce results using models WITHOUT aggregate shocks.

- "WithAggShock" folder contains various files which produce results using models with Krusell-Smith aggregate shocks. 

- "WithAggPermShocks" folder contains various files which produce results using models with aggregate PERMANENT shocks (FBS agg shocks). 

- "Results" folder stores various simulation results 


- DoAll.nb runs all the models with/without aggregate shocks (and produces tables for the paper automatically) 

- DoAll_cstKS.nb runs the files for cstKS project (both for longer WP version and Economics Letters version)
